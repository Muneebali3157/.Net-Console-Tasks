## 📌 Overview

This repository contains **C# tasks** focusing on C# basics and Object-Oriented Programming (OOP) principles.

### 🚀 Projects Included

1. **Task 1: Console Calculator** – Menu-driven basic arithmetic calculator.
2. **Task 2: Number Properties Toolkit** – Utility program for analyzing number properties ( eg prime, palindrom , armstrong , PrintFactorial etc)
3. **Task 3: Student Class** – Fundamentals of OOP (Encapsulation, Properties, Methods).
4. **Task 4: Class Inheritance** – Realizing relationships using `Person`, `Employee`, and `Student` base/derived classes.
5. **Task 5: Mini Project – Employee Payroll System** – Menu-driven console app that calculates basic salary, allowances, deductions, and prints aligned salary slips.
